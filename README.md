## 소개  

【에스피온(SP:ON)】

에스피온의 리더이자, 요새방벽단 단장 "소소와플" 의 노래책 입니다.

KPOP / JPOP / POP 등 

실시간 방송(치지직)에서 노래 컨텐츠를 할 때,
해당 노래책을 참고하여 불러줬음하는 노래를 추천해주세요!

- 목 컨디션에 따라 부르기 어렵다고 생각되는 노래는 거절될 수 있습니다.
- 노래 신청을 할 때, "제목-가수"를 꼭 적어주세요.
- 노래책에 없는 노래도 "후원으로 연습"시킬 수 있습니다.
   ▷ 다음 노래방송에 순서대로 부릅니다.
   ▷ 노래 예약이 밀려있는 경우, 밀릴 수 있습니다.
- 본 방송인이 거절의사를 밝혔음해도 불구하고 강요하는 경우, 벤 당할 수 있습니다.
- 노래평가 및 훈수는 금지 입니다. / 예쁜 채팅 부탁드리겠습니다.
- 자세한 규칙은 "디스코드"를 참고해주세요!
